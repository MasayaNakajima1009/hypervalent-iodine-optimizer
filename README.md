# Hypervalent Iodine Optimizer

A Google Colab workflow for geometry optimization of hypervalent iodine compounds using a machine-learning potential.

Developed by

Masaya Nakajima
Associate Professor
The University of Tokyo

---

## Overview

This repository provides a demonstration workflow for geometry optimization of hypervalent iodine compounds.

The notebook accepts an XYZ structure as input and returns an optimized XYZ structure using a pretrained machine-learning potential.

The pretrained model is not included in this repository and will be released after acceptance of the corresponding research manuscript.

---

## Repository Contents

* `HVI_Optimizer.ipynb`

  Google Colab notebook for geometry optimization.

* `Sample_input.xyz`

  Example input structure.

* `Sample_input_optimized.xyz`

  Example optimized structure generated by the workflow.

---

## Usage

1. Open the notebook in Google Colab.
2. Upload an XYZ structure.
3. Specify the path to the pretrained model.
4. Run geometry optimization.
5. Download the optimized XYZ structure.

---

## Intended Use

This repository is intended as a demonstration of the workflow used for machine-learning-assisted geometry optimization of hypervalent iodine compounds.

The notebook is provided for educational and research purposes.
